## Como rodar aplicação

Entre na pasta do diretório  <br />
cd covid-map <br />

Instale as dependências   <br />
npm install <br />

Escute a aplicação com : <br />
npm start <br />

Acesse [http://localhost:3000](http://localhost:3000) para visualizar no seu naveador.<br />
